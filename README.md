# Guia-Gourmet-Intergaláctico
Uma aplicação voltada para a gastronomia do universo.
 * **Dependências:**
     * express
    ``` npm install express ```
    * nodemon
    ``` npm install nodemon ```
    * pg
    ``` npm install pg ```
    * cors
    ``` npm install cors ```
    * dotenv
    ``` npm install dotenv ```
    * body-parser
    ``` npm install npm install body-parser ```
